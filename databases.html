My React Chat Application

During my computer science journey, I have had the opportunity to work on several different projects that have enhanced my technological skills.

A particular project that I wanted to better enhance, and further perfect is ‘My Chat Application’. This artifact, which I created about 8 months ago, is the only project I have created for my colleagues. This application was built using MongoDB, Socket.io and Node.js. It was not a production level application, but rather, assisted with getting familiar with these types of technologies in an amusing and creative way and it allowed our team to communicate with one another.

This application was meant to enhance my knowledge and skills in web clients, servers and databases
